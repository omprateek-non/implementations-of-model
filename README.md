This project is a Student Performance Prediction System built using Machine Learning and deployed with Streamlit.
It predicts a student's academic grade class based on behavioral, academic, and socio-demographic features.

Unlike basic ML demos, this project focuses on real-world usability, allowing users to input data through an interactive UI and get instant predictions.

##Model Details
Algorithm: K-Nearest Neighbors (KNN)
Accuracy: ~95% (evaluated on test data)
Features Used:
Age
Gender
Ethnicity
Parental Education
Study Time Weekly
Absences
Tutoring
Parental Support
Extracurricular Activities
Sports
Music
Volunteering
GPA
